# 🧠 Artificial Intelligence System for Correcting and Resulting

An intelligent web-based system that automates the evaluation of handwritten or typed answer scripts using OCR and Natural Language Processing (NLP).

---

## 🚀 Project Overview

The AI-Based Paper Correction System is designed to reduce manual effort in evaluating answer sheets by automatically extracting text and comparing it with model answers to generate scores.

This system is especially useful for:
- Educational institutions
- Online exams
- Automated assessment platforms

---

## ✨ Features

- 📄 Upload student answer sheets (PDF/Image)
- 🔍 Extract text using OCR (Tesseract)
- 🧠 Evaluate answers using NLP techniques
- 📊 Generate marks based on similarity
- 🌐 User-friendly web interface (Flask)
- ⚡ Fast and automated evaluation

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **OCR:** Tesseract OCR  
- **NLP:** NLTK / Scikit-learn  
- **Frontend:** HTML, CSS  
- **Libraries:** NumPy, Pandas
